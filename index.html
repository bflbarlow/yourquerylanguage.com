YourQL

Your data. Your questions. Your answers.

Chat with your database. A free, simple, and open source data harness desktop application that connects you and large language models to your data.

Work in progress — github.com/bflbarlow/YourQL

How it works

You start a discussion, ask a question in plain English, and the Discussion Engine takes over. It reads your database schema, sends it to the LLM along with your question, executes the SQL that comes back, and shows you the results — all within a chat interface.

Question → LLM → SQL → Results

What it can do

  • Natural language to SQL via any supported LLM
  • Schema introspection — tables, columns, indexes, foreign keys
  • Exploration mode — the LLM can run intermediate queries for context
  • Read-only safety on all exploration queries
  • Self-correction — SQL errors are fed back to the LLM
  • Persistent conversations and full message history
